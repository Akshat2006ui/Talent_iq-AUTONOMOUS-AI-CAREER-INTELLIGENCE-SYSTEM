# Developer Photo Setup

To display Akshat Kapoor's photo on the About page:

1. Save the developer photo as `developer-photo.jpg` in the `frontend/public/` directory
2. The photo should be a professional headshot
3. Recommended dimensions: 400x400 pixels or larger (square format works best)
4. Format: JPG or PNG

The photo will be displayed on the About page at `/about` route.

If the photo is not found, the page will display initials "AK" as a fallback.
